# Mystery Hangman Arena
# Description
Mystery Hangman Arena is an interactive word guessing game developed in Python.
# Features
1.Three difficulty levels
2.Hint system
3.Score tracking
4.Win/Loss statistics
5.Previous guessed letters
6.Play again option
# Technologies used
1.Python
2.Random module
3.Functions
4.Lists
5.Loops
6.File Handling
# How to Run
1.Open terminal.
2.Navigate to project folder.
3.Execute using a command :python main.py
